## 📖 Sobre

O **Portal de Notícias** é uma página estática desenvolvida com HTML e CSS, inspirada em um portal de tecnologia.

O projeto apresenta notícias em destaque, conteúdos mais lidos da semana, artigos sobre inteligência artificial, anúncios e matérias complementares.

O layout foi adaptado para diferentes tamanhos de tela, oferecendo uma boa experiência de navegação em computadores, tablets e celulares.

## 🚀 Tecnologias

- HTML5
- CSS3
- CSS Grid
- Flexbox
- Media Queries
- CSS Nesting

## 📱 Responsividade

O projeto possui diferentes adaptações de layout conforme o espaço disponível:

- Reorganização das colunas em telas menores
- Cards de destaque adaptáveis
- Notícias mais lidas distribuídas em duas ou quatro colunas
- Navegação de categorias com rolagem horizontal
- Textos do cabeçalho ocultados em celulares
- Imagens redimensionadas sem deformação com `object-fit`
- Limitação de textos com reticências utilizando `line-clamp`
- Seções de Inteligência Artificial adaptadas para desktop, tablet e mobile
